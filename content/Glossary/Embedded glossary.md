### Embedded technical word encountered:

HAL: Hardware Abstraction Layout, it's an API that ease the embedded development.
GPIO: General Purpose Input/Output, use to communicate with the components of the board
ADCs: Analog to Digital Converters
RCC: Reset and Clock Control, control clock and peripheral reset
AHB1: Advanced High-Performance Bus 1
ENR: Enable Register
ODR: Output Data Register, use to control the output state of each GPIO pin

#### Communication interfaces
UART: Universal Asynchronous Receiver-Transmitter,
He's a communication device use to send and receive data between two system.


SPI: Serial Peripherical Interface, synchronous and use to communicate with others peripherical

I2C: Inter-integrated Circuit, synchronous and use to connect many devices on the same bus with less cables
